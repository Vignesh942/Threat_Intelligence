# AI-Powered Real-Time Threat Feed Aggregator (CLI Tool)

**Real-time threat intelligence at your fingertips**

This lightweight Python command-line tool solves a common problem for security analysts: gathering threat intelligence from multiple sources without jumping across websites. It aggregates results, explains the risks, and saves them for audits or investigations.

---

## Features

1. **Fetches intel from multiple sources at once**  
   Supports:  
   - OTX (Open Threat Exchange)  
   - VirusTotal  
   - AbuseIPDB  
   - URLScan  
   - WHOIS  

   One command gives you all the information you need about domains, URLs, or IPs.


    **Uses Gemini Flash to summarize risk**  
   - Gemini Flash 2.0 generates a 0-10 threat score  
   - Provides concise summaries and actionable recommendations  
   - Eliminates the need to manually analyze each source


3. **Exports results for audits and investigations**  
   - Clean JSON files compatible with SIEMs or custom investigation tools  
   - Easy to save, search, and share
